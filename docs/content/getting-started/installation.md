---
title: Installation
description: Get started with Material Yu in your Nuxt application.
---

To get started, install Material Yu using a package manager.

1. Install the Material Yu package

   ```bash [pnpm]
   pnpm add @yurai/material-yu
   ```

2. Add the Material Yu module to your `nuxt.config.ts`

   ```ts [nuxt.config.ts]
   export default defineNuxtConfig({
     modules: ['@yurai/material-yu']
   })
   ```
