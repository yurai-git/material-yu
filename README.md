**Note:** Material Yu is an independent project and is not affiliated with Google or the Material Design team.

## Local development

To begin local development, follow the following steps:

1. Clone the repository to your local machine:

   ```bash
   git clone https://github.com/yurai-git/material-yu.git
   cd material-yu
   ```

1. Install pnpm:

   ```bash
   npm install -g pnpm
   ```

1. Install the dependencies:

   ```bash
   pnpm install
   ```

1. Start the development server:

   ```bash
   pnpm dev
   ```
